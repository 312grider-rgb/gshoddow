# 🎓 SkillStream - Live Classroom

A real-time virtual classroom with video conferencing, chat, and an AI assistant.

## ✨ Features

- 🎥 **Video Conferencing** - Peer-to-peer video with WebRTC
- 💬 **Real-time Chat** - Live messaging with the class
- 🤖 **AI Assistant** - Ask questions and get instant answers
- 👥 **Participant List** - See who's in the room
- 📱 **Mobile Friendly** - Works on phones and tablets
- 🔒 **No Sign-up Required** - Just enter your name and join

## 🚀 Quick Start

1. Open `index.html` in your browser
2. Enter your name
3. Create a room or join an existing one
4. Allow camera and microphone access
5. Start teaching or learning!

## 🎯 How to Use

### Join a Session
- Enter your name
- Either:
  - Create a new room (leave room code blank)
  - Enter an existing room code
  - Click a quick-join button

### During a Session
- **Chat**: Type messages in the chat panel
- **AI**: Ask the AI assistant questions
- **Video**: Toggle your camera and mic on/off
- **Reactions**: Send emoji reactions to the class

## 📦 Deployment

### Deploy to GitHub Pages

1. Upload `index.html` to your repository
2. Go to Settings → Pages
3. Select "main" branch as source
4. Your site will be live at: `https://YOUR_USERNAME.github.io`

### Deploy to Render

1. Create a new Static Site on Render
2. Connect your repository
3. Set Publish Directory to `.` (root)
4. Deploy

## 💻 Tech Stack

- **Frontend**: Vanilla JS, HTML5, CSS3
- **Video**: PeerJS (WebRTC)
- **AI**: Built-in mock responses (no API key needed)
- **Hosting**: GitHub Pages or any static host
